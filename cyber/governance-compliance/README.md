# PHOENIX Governance & Compliance Documentation

This repository contains governance and compliance documentation developed for the PHOENIX Cyber Resilience Project as part of SIT374 / SIT764 – Team Project A at Deakin University.

The documentation focuses on cyber resilience, governance frameworks, compliance management, secure operational practices, and cybersecurity risk mitigation within the PHOENIX project environment.

# Compliance & Standards Alignment

The governance framework aligns with recognised cybersecurity standards and industry best practices, including:

- ISO/IEC 27001
- NIST Cybersecurity Framework
- ACSC Essential Eight
- OWASP Security Guidelines
- Australian Privacy Principles (APP)

---

# Project Context

PHOENIX is a cyber resilience platform designed to integrate natural disaster intelligence with cyber threat monitoring and response capabilities. The project focuses on improving governance, operational resilience, threat awareness, and secure decision-making during disaster-related cyber incidents.

The Governance & Compliance stream supports the project through structured governance frameworks, operational procedures, compliance practices, and cybersecurity documentation.
